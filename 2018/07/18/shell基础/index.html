  2、设置周期性计划任务
        * * * * * python3 /home/tarena/python/01.py
        * * * * * sh /home/tarena/sh/01.sh
        分时日月周
        分 ：0-59
        时 ：0-23
        日 ：1-31
        月 ：1-12
        周 ：0-6

        * ：代表所有可能值
        , ：指定多个时间点  
            * * 1,5 * * 每月的1日和5日两天
        / ：指定时间间隔频率
            */10 * * * * 表示每10分钟执行一次命令
        - ：指定一个时间段
            0 0-6/1 * * * 表示0点-6点没小时执行一次命令
      3、每分钟执行一次 01.py 文件
        * * * * * python3 .../01.py
      4、每小时的第3分钟和第15分钟执行
        3,15 * * * * python3 .../01.py
      5、每周六、周日的0点执行一个 01.py 文件
        0 0 * * 6,0 python3 ..../01.py
      6、每天18:00到23:00之间每小时执行 01.py 文件
        0 18-23/1 * * * python3 .../01.py

5、until循环
      1、特点
        条件判断不成立时执行循环体,成立时循环结束
      2、把176.136.10.10-15内不在线的IP输出来
        #!/bin/bash
        x=10
        until [ $x -gt 15 ]
        do
            ping -c 2 176.136.10.$x &> /dev/null
            if [ $? -ne 0 ];then
                echo "176.136.10.$x"
            fi
            let x++
        done

4、sed(文本处理工具-Linux)
      1、打印 p
        sed -n 'p' a.txt ##打印文件中所有内容
        sed -n '1p' a.txt  ##打印第1行
        sed -n '1,4p' a.txt ##打印1-4行
        sed -n '1p;5p' a.txt ##打印第1行和第5行
      2、删除 d
        sed -i '1d' a.txt ##删除第1行内容
        sed -i '$d' a.txt ##删除最后1行内容
        sed -i '1,3d' a.txt ##删除1-3行
        sed -i '1d;3d' a.txt ##删除第1行和第3行
      3、替换(整行替换) c
        sed -i 'c内容' a.txt
        sed -i '条件c内容' a.txt
        sed -i '1,3c将进酒' a.txt ##1-3行替换为将进酒
        sed -i -e '1c李白' -e '2c杜甫' a.txt 
      4、添加 a/i 
        a : 在当前处理行的下方添加新行
        i ：在当前处理行的上方添加新行
        sed -i 'atiger' a.txt ##所有行下方都添加tiger
        sed -i '1alion' a.txt ##第一行下方添加
        sed -i '1irabbit' a.txt ##第一行上方添加
      5、替换 s
        sed -i 条件s/旧内容/新内容/选项
        选项 ：g 表示全局替换 i 忽略字母大小写替换
        sed -i '1,3s/6/A/g' a.txt  ##1-3行中6全部变为8
        sed -i 's/[0-9]/B/g' a.txt  ##文本中所有数字B
        sed -i 's/[0-9]\+/C/g' a.txt ##把文本中连续出现的数字替换为C

     ** 加 g 表示全部替换,不加 g 只会替换指定行中的第一个
